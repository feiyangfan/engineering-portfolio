URL Shortener

A simple URL shortener I am trying to build. Covert URLs to short URLs.

Github Link

Project Overview

The inspiration for this project came from when I used the share function on Youtube. It shortened the URL in the URL address bar, and gave me a short URL that still direct me to the video. I wanted to know how that works and build one myself.

What I learned:
The way URL shortener works is that when a user enter a URL that is to be shortened, we send it to our database and automatically generate a corresponding unique key, which is going to be shortened URL, and store them together in our database. Then when a user accesses a shortened URL, we find the corresponding original URL in our database and redirect the user to it.

Created a responsive website using Express and MongoDB, deployed on Heroku
Implemented URL shortening using UUID and URL forwarding
Provided a list of all shortened URLs for user to reuse
