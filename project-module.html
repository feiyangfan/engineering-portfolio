Github Link

Project Overview

This is a project that was inspired from the process of making myself a personal website. During the process, I have to constantly modify the html file. It becomes a chore and could be energy draining. Therefore, I wanted to build a interface that is easy to use. This interface is easy to navigate, intuitive to use, and does not require any knowledge in web development.

Created and maintained a responsive web application using Express and MongoDB
Implemented a modular information management interface
Implemented user authentication with username and password
